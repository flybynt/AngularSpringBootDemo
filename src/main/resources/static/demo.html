The AngularJS MVC demo lists all of the data in the H2 database utilizing two different JAVA Springboot REST API approaches. The data can be filtered by entering values in the appropriate fields for either the @RepositoryRestResource or @RestController portions of the app which can be accessed above. Additionally, new data rows can be easily entered to the database by entering data in the fields of the Add portion of the app which can also be accessed above
