AngularJS MVC - List Data

This portion of the AngularJS MVC demo lists all of the data in the H2 database utilizing the REST interface to JAVA Springboot repository interface. The data can be filtered by entering values in the appropriate field for the data set below. To edit data for a displayed record, click anywhere in that row.
